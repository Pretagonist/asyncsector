Async Sector
===============================

Python package for Sector Alarm using asyncio and aiohttp

Installation / Usage
--------------------

To install use pip:

    $ pip install asyncsector


Or clone the repo:

    $ git clone https://github.com/mgejke/asyncsector.git
    $ python setup.py install

Run the script:

    $ asyncsector <alarmid> <username> <password>
