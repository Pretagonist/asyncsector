Async Sector
===============================

version number: 0.0.1
author: Martin Gejke

Overview
--------

Asynchronous package for Sector Alarm

Installation / Usage
--------------------

To install use pip:

    $ pip install asyncsector


Or clone the repo:

    $ git clone https://github.com/mgejke/asyncsector.git
    $ python setup.py install
